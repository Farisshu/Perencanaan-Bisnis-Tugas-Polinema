## D. GLOSARIUM ISTILAH TEKNIS & FINANSIAL

### Istilah Teknis Embedded System

| **Istilah** | **Kepanjangan** | **Definisi** |
|-------------|-----------------|--------------|
| MCU | Microcontroller Unit | Chip komputer mini dalam satu IC |
| GPIO | General Purpose Input/Output | Pin yang bisa dikonfigurasi sebagai input/output |
| PWM | Pulse Width Modulation | Teknik modulasi sinyal digital untuk simulasi analog |
| ADC | Analog-to-Digital Converter | Mengubah sinyal analog ke digital |
| UART | Universal Asynchronous Receiver-Transmitter | Protokol komunikasi serial |
| SPI | Serial Peripheral Interface | Protokol komunikasi synchronous serial |
| I2C | Inter-Integrated Circuit | Protokol komunikasi two-wire serial |
| CAN Bus | Controller Area Network | Protokol komunikasi untuk otomotif/industrial |
| FreeRTOS | Free Real-Time Operating System | OS real-time open-source untuk embedded |
| IoT | Internet of Things | Jaringan perangkat fisik yang terhubung internet |
| PCB | Printed Circuit Board | Papan sirkuit cetak untuk komponen elektronik |
| BOM | Bill of Materials | Daftar komponen untuk produksi |
| HPP | Harga Pokok Penjualan | Total biaya untuk memproduksi satu unit produk |

### Istilah Finansial

| **Istilah** | **Definisi** | **Formula** |
|-------------|--------------|-------------|
| BEP | Break-Even Point | Titik impas dimana revenue = total cost |
| ROI | Return on Investment | Pengembalian investasi (%) | (Laba Bersih ÷ Modal) × 100% |
| NPV | Net Present Value | Nilai bersih sekarang dari arus kas masa depan | Σ(CF/(1+r)^t) - Initial Investment |
| IRR | Internal Rate of Return | Tingkat diskonto yang membuat NPV = 0 | - |
| HPP | Harga Pokok Penjualan | Biaya langsung untuk memproduksi produk | Bahan baku + Tenaga kerja + Overhead |
| Margin Kotor | Selisih harga jual dan HPP | (Harga Jual - HPP) ÷ Harga Jual × 100% |
| Cash Flow | Arus Kas | Masuk dan keluarnya kas dalam periode tertentu | Cash In - Cash Out |
| Working Capital | Modal Kerja | Dana untuk operasional harian | Aset Lancar - Kewajiban Lancar |
| Depresiasi | Penyusutan | Penurunan nilai aset tetap seiring waktu | (Cost - Salvage) ÷ Useful Life |
| WACC | Weighted Average Cost of Capital | Biaya modal rata-rata tertimbang | (E/V×Re) + (D/V×Rd×(1-T)) |
