## B. RUMUS-RUMUS KEUANGAN

### B.1 Break-Even Point (BEP)

```
┌─────────────────────────────────────────────────────────────┐
│                    BEP FORMULAS                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  BEP (Unit)    = Fixed Cost ÷ (Price - Variable Cost)      │
│                                                              │
│  BEP (Rupiah)  = Fixed Cost ÷ Contribution Margin Ratio    │
│                                                              │
│  Where:                                                     │
│  • Contribution Margin = Price - Variable Cost             │
│  • CM Ratio = Contribution Margin ÷ Price                  │
│                                                             │
│  Contoh EduKit IoT:                                         │
│  BEP unit = Rp 30.000.000 ÷ (Rp 275.000 - Rp 185.000)      │
│           = Rp 30.000.000 ÷ Rp 90.000                      │
│           = 333 unit/tahun                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### B.2 Return on Investment (ROI)

```
┌─────────────────────────────────────────────────────────────┐
│                    ROI FORMULAS                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ROI (%) = (Net Profit ÷ Total Investment) × 100%          │
│                                                             │
│  Annualized ROI = [(Ending Value ÷ Beginning Value)^       │
│                     (365 ÷ Days)] - 1                       │
│                                                             │
│  Cumulative ROI = Σ(Annual Net Profit) ÷ Investment        │
│                                                             │
│  Contoh EduKit IoT Tahun 3:                                 │
│  ROI = (Rp 74.583.750 ÷ Rp 100.000.000) × 100%            │
│      = 74.6%                                                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### B.3 Depresiasi (Penyusutan)

```
┌─────────────────────────────────────────────────────────────┐
│              DEPRECIATION METHODS                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  1. Straight-Line Method:                                   │
│     Depreciation/year = (Cost - Salvage Value) ÷ Life      │
│                                                             │
│     Contoh Peralatan:                                       │
│     = (Rp 32.800.000 - Rp 0) ÷ 5 tahun                     │
│     = Rp 6.560.000/tahun                                   │
│                                                             │
│  2. Double-Declining Balance:                               │
│     Rate = 2 ÷ Useful Life                                 │
│     Depreciation = Book Value × Rate                       │
│                                                             │
│  3. Units of Production:                                    │
│     Depreciation/unit = (Cost - Salvage) ÷ Total Units     │
│     Annual Depr = Depreciation/unit × Units Produced       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### B.4 Net Present Value (NPV)

```
┌─────────────────────────────────────────────────────────────┐
│                    NPV FORMULA                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  NPV = Σ [Cash Flow_t ÷ (1 + r)^t] - Initial Investment   │
│                                                              │
│  Where:                                                     │
│  • t = period (year)                                        │
│  • r = discount rate (cost of capital)                     │
│                                                              │
│  Decision Rule:                                             │
│  • NPV > 0  → Accept (layak)                               │
│  • NPV < 0  → Reject (tidak layak)                         │
│  • NPV = 0  → Indifferent                                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### B.5 Internal Rate of Return (IRR)

```
┌─────────────────────────────────────────────────────────────┐
│                    IRR CONCEPT                              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  IRR adalah discount rate yang membuat NPV = 0             │
│                                                              │
│  0 = Σ [CF_t ÷ (1 + IRR)^t] - Initial Investment           │
│                                                              │
│  Decision Rule:                                             │
│  • IRR > Cost of Capital → Accept                          │
│  • IRR < Cost of Capital → Reject                          │
│                                                              │
│  EduKit IoT: IRR = 52% (>> 10% WACC) → ACCEPT              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## G. GLOSSARY

| **Istilah** | **Definisi** |
|-------------|--------------|
| **BEP** | Break-Even Point - titik dimana pendapatan = biaya total |
| **HPP** | Harga Pokok Penjualan - direct cost untuk memproduksi 1 unit |
| **ROI** | Return on Investment - persentase keuntungan dari investasi |
| **NPV** | Net Present Value - nilai sekarang dari arus kas masa depan |
| **IRR** | Internal Rate of Return - tingkat diskonto yang membuat NPV = 0 |
| **WACC** | Weighted Average Cost of Capital - biaya modal rata-rata tertimbang |
| **EBITDA** | Earnings Before Interest, Taxes, Depreciation, and Amortization |
| **CapEx** | Capital Expenditure - pengeluaran untuk aset tetap |
| **OpEx** | Operational Expenditure - pengeluaran operasional harian |
| **Cash Runway** | Berapa lama bisnis bisa bertahan dengan cash saat ini |
